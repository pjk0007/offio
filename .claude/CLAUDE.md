## 기술 스택

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS + shadcn/ui
- Drizzle ORM + PostgreSQL
- Cloudflare R2 (스크린샷 저장)
- NextAuth + JWT 인증
