## 영업 포인트

### 핵심 메시지

```
"Offio 도입하면 비용이 아니라 수익입니다"

구축비 → 정부가 절반 지원
월비용 → 장려금으로 충당하고도 남음
```

### 플랜별 멘트

**Lite 추천 고객** (소규모, 단순 관리)
> "관리자-근무자 구조로 간단하게 재택근무 증빙을 관리하세요."

**Standard 추천 고객** (중규모, 팀 단위 관리)
> "팀장이 자기 팀의 근태를 직접 승인! 지원금 증빙 리포트도 자동 생성됩니다."

**Enterprise 추천 고객** (HR 없음 or 통합 원함)
> "휴가, 조직관리까지 Offio 하나로 해결하세요. 별도 HR 시스템 불필요합니다."
