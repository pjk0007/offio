# Offio 화면 설계

## 화면 구조

```
[로그인] ─┬─▶ [근무자 대시보드] ─── 근무 기록 조회/편집/제출
          │
          └─▶ [관리자 대시보드] ─── 팀 현황/승인/리포트
```

---

## 공통

### 로그인 페이지

```
┌─────────────────────────────────────┐
│                                     │
│            [Offio 로고]             │
│                                     │
│   ┌─────────────────────────────┐   │
│   │ 이메일                      │   │
│   └─────────────────────────────┘   │
│   ┌─────────────────────────────┐   │
│   │ 비밀번호                    │   │
│   └─────────────────────────────┘   │
│                                     │
│   [━━━━━━━ 로그인 ━━━━━━━]         │
│                                     │
│   비밀번호 찾기                     │
│                                     │
└─────────────────────────────────────┘
```

**기능:**
- 이메일/비밀번호 로그인
- 비밀번호 찾기 링크
- 로그인 후 역할(role)에 따라 대시보드 분기

**상태 배지:**
- `기록중` - 파란색 (현재 데스크톱 앱에서 기록 중)

- `제출됨` - 주황색 (관리자 검토 대기)
- `승인됨` - 초록색
- `반려됨` - 빨간색

## 화면 흐름

### 근무자 플로우

```
[로그인] → [대시보드] → [근무기록 목록] → [상세/편집]
                                              ↓
                                         [제출 미리보기]
                                              ↓
                                           [제출]
```

### 관리자 플로우

```
[로그인] → [대시보드] → [승인대기 목록] → [상세 확인]
               │                              ↓
               │                      [승인] or [반려]
               │
               ├→ [팀원 관리]
               └→ [월간 리포트]
```

## 반응형 고려사항

| 화면 | 데스크톱 | 태블릿 | 모바일 |
|------|----------|--------|--------|
| 대시보드 | 4열 카드 | 2열 카드 | 1열 스택 |
| 타임라인 | 전체 표시 | 전체 표시 | 가로 스크롤 |
| 스크린샷 | 6열 그리드 | 4열 그리드 | 2열 그리드 |
| 테이블 | 전체 컬럼 | 주요 컬럼 | 카드 형태 |
