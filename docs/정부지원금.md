# 2025년 유연근무 장려금 정리

## 개요

고용노동부에서 일·생활 균형 문화 확산을 위해 유연근무제 도입 기업에 지원금 지급

## 지원 대상

- **우선지원대상기업 (중소기업)**
- **중견기업**
- 대기업은 해당 없음

## 지원 금액

### 재택/원격 근무

| 월 활용일 | 월 지원금 | 연간 최대 |
|----------|----------|----------|
| 4~7일 | 15만원 | 180만원 |
| 8~11일 | 20만원 | 240만원 |
| 12일 이상 | 30만원 | 360만원 |

### 육아기 근로자 (12세 이하 자녀)

- 일반 근로자 대비 **2배 지원**
- 연간 최대 **720만원**

### 지원 한도

- 피보험자수의 30% (최대 70명)

## 지원 요건

### 공통 요건

1. 주당 소정근로시간 35~40시간
2. 근로계약서에 변경된 근로시간/장소 규정
3. **전자적·기계적 방식으로 출퇴근 시각 기록·관리**

### 근태관리 인정 방식

- 전자카드, 지문인식, 타임레코드
- **그룹웨어** ← Offio 해당
- 재택근무 한정: 이메일, 카카오톡 캡처도 가능

### 제출 서류

- 유연근무 활용 전·후 근로계약서
- 월별 임금대장, 임금 지급 증빙
- 취업규칙
- **출퇴근 기록 증빙 서류** ← Offio 리포트로 대체 가능

## 신청 절차

```
사업계획서 제출 → 고용센터 심사·승인 → 제도 도입·활용 → 지원금 신청 → 지급
```

- **신청처**: 고용24 (www.work24.go.kr)
- **신청 주기**: 3개월 단위
- **신청 기한**: 유연근무 활용일 다음 달부터 12개월 이내

## 문의처

- 고용노동부 고객상담센터: **1350** (3→6 사업주지원금)
- 사업장 소재지 관할 고용센터

## Offio 연계 전략

### 정부 요건 vs Offio 기능

| 정부 요건 | Offio 제공 기능 |
|----------|----------------|
| 출퇴근 시각 기록 | 앱 시작/종료 시간 자동 기록 |
| 전자적 방식 | 서버 자동 저장 |
| 실근로시간 증빙 | 키보드/마우스 활동 데이터 |
| 재택 여부 확인 | IP주소, 스크린샷 |

### 마케팅 포인트

```
"Offio 도입 → 지원금 수령 → 비용보다 이득"

예시) 직원 10명, 월 12일 이상 재택
- Offio 비용: ?만원
- 지원금: 30만원 × 10명 = 월 300만원
```

### 필요 기능

- [ ] 월별 재택근무 일수 자동 집계
- [ ] 출퇴근 기록 리포트 출력
- [ ] 정부 제출 양식 맞춤 export
