# Offio

**재택근무 증빙 기반 근태관리 솔루션**

## 핵심 키워드

- **모니터링** - 키보드/마우스 활동, 프로그램 사용, 스크린샷 자동 수집
- **프라이버시** - 근무자가 직접 편집 후 제출 (감시 아닌 증빙)
- **정부지원금** - 유연근무 장려금 + 인프라 구축비 연계

## 왜 Offio인가?

| 기존 서비스 | 문제점 | Offio |
|------------|--------|-------|
| flex, 시프티 등 | 자기 신고 → 신뢰도 낮음 | 실제 활동 데이터 증빙 |
| Hubstaff, Time Doctor | 실시간 감시 → 직원 반발 | 근무자가 편집권 보유 |
| 수기 관리 | 지원금 증빙 어려움 | 자동 리포트 생성 |

## 구성

```
[데스크톱 앱] ─────▶ [서버] ◀───── [웹 대시보드]
  근무자 PC            데이터 저장        관리자/근무자
  - 활동 수집          - 리포트 생성      - 현황 조회
  - 스크린샷           - 지원금 증빙      - 편집/제출
```
