# Offio 기능 정의

## 시스템 구성

```
[데스크톱 앱] ───▶ [서버/API] ◀─── [웹 대시보드]
   근무자 PC                          관리자/근무자
```

---

## 1. 데스크톱 앱 (기존 개발 완료)

### 모니터링 기능

| 기능 | 설명 |
|------|------|
| 키보드 활동 | 키 입력 횟수 (내용 X) |
| 마우스 활동 | 클릭, 이동 횟수 |
| 포커싱 프로그램 | 현재 사용 중인 프로그램명 |
| 스크린샷 | 1분마다 캡처 (간격 설정 가능) |
| 서버 전송 | 수집 데이터 실시간 전송 |

### 근무 제어

| 기능 | 설명 |
|------|------|
| 근무 시작 | 버튼 클릭으로 모니터링 시작 |
| 근무 종료 | 버튼 클릭으로 모니터링 종료 |
| 일시 정지 | 휴식/외출 시 일시 중단 |

---

## 2. 웹 대시보드 - 근무자용 (개발 필요)

### 근무 기록 조회

| 기능 | 설명 |
|------|------|
| 일별 타임라인 | 시간대별 활동 내역 시각화 |
| 스크린샷 갤러리 | 캡처된 스크린샷 목록 |
| 프로그램 사용 통계 | 어떤 프로그램을 얼마나 사용했는지 |
| 주간/월간 요약 | 총 근무시간, 평균 등 |

### 근무 기록 편집 (핵심 기능)

| 기능 | 설명 |
|------|------|
| 구간 제외 | 점심시간, 개인시간 등 제외 처리 |
| 스크린샷 삭제 | 민감한 스크린샷 제거 |
| 시간 조정 | 출퇴근 시간 수정 요청 |
| 메모 추가 | 특이사항 기록 |

### 제출

| 기능 | 설명 |
|------|------|
| 일일 제출 | 편집 완료 후 관리자에게 제출 |
| 제출 전 미리보기 | 관리자에게 보여질 내용 확인 |
| 제출 이력 | 과거 제출 내역 조회 |

---

## 3. 웹 대시보드 - 관리자용 (개발 필요)

### 팀 현황

| 기능 | 설명 |
|------|------|
| 실시간 현황 | 현재 근무 중인 팀원 목록 |
| 오늘 근태 요약 | 출근/미출근 현황 |
| 주간/월간 통계 | 팀 전체 근무시간 집계 |

### 근무 기록 확인

| 기능 | 설명 |
|------|------|
| 제출된 기록 조회 | 팀원이 제출한 근무 기록 |
| 타임라인 뷰 | 시간대별 활동 내역 |
| 스크린샷 뷰 | 캡처된 화면 확인 |

### 승인/관리

| 기능 | 설명 |
|------|------|
| 근무 기록 승인 | 제출된 기록 승인/반려 |
| 코멘트 | 피드백 작성 |
| 수정 요청 | 재제출 요청 |

### 리포트

| 기능 | 설명 |
|------|------|
| 월간 근태 리포트 | 팀원별 근무 현황 |
| 지원금 증빙 리포트 | 정부 제출용 양식 출력 |
| 엑셀 다운로드 | 데이터 내보내기 |

## 4. 시스템 관리 (개발 필요)

### 회사/조직 설정

| 기능 | 설명 | 플랜 |
|------|------|------|
| 회사 정보 | 회사명, 로고 등 | 전체 |
| 직원 기본 관리 | 직원 추가/삭제 | 전체 |
| 부서 관리 | 부서 생성/수정/삭제 | Standard+ |
| 팀장 역할 | 팀장이 자기 부서 근태 승인 | Standard+ |
| 조직도 관리 | 조직 구조 시각화, 부서 계층 | Enterprise |
| 직원 상세 정보 | 연락처, 사번, 직급, 입사일, 연차 관리 | Enterprise |

### 근무 정책 설정

| 기능 | 설명 |
|------|------|
| 근무 시간 | 출퇴근 기준 시간 |
| 유연근무제 | 코어타임 설정 |
| 스크린샷 간격 | 캡처 주기 설정 |

## MVP 범위

### Phase 1 (MVP) - Lite + Standard 공통

- [x] 데스크톱 앱 (기존)
- [ ] 근무자 웹: 기록 조회, 편집, 제출
- [ ] 관리자 웹: 팀 현황, 기록 확인, 승인
- [ ] 기본 리포트 (월간 근무시간)

### Phase 2 - Standard 차별화

- [ ] 부서 관리
- [ ] 팀장 역할 (자기 부서 근태 승인/리포트)
- [ ] 지원금 증빙 리포트
- [ ] 엑셀 다운로드

### Phase 3 - Enterprise 기능

- [ ] 직원 상세 정보 관리 (연락처, 사번, 직급, 입사일, 연차 등)
- [ ] 휴가 신청/승인
- [ ] 조직도 관리 (부서 계층, 시각화)
- [ ] 근무 정책 설정
- [ ] 고급 통계/분석
