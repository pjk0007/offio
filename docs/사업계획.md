# Offio 사업계획

## 타겟 고객

### 1차 타겟: IT/소프트웨어 중소기업

| 조건 | 이유 |
|------|------|
| 업종: IT/소프트웨어 | 재택근무 비율 85%로 가장 높음 |
| 규모: 10~50명 | 정부 지원금 대상, 의사결정 빠름 |
| 형태: 중소기업 | 유연근무 장려금 + 인프라 구축비 대상 |

### 재택근무 비율 높은 업종 순위

1. IT/소프트웨어
2. 디자인/크리에이티브
3. 마케팅/광고
4. 콘텐츠/미디어
5. 전문서비스 (회계, 법률, 컨설팅)

## 차별점

### 경쟁사 비교

| 구분 | 기존 근태관리 (flex, 시프티) | 감시형 (Hubstaff, Time Doctor) | **Offio** |
|------|---------------------------|------------------------------|-----------|
| 기록 방식 | 자기 신고 (버튼 클릭) | 실시간 감시 | 활동 증빙 |
| 신뢰도 | 낮음 (조작 가능) | 높음 | 높음 |
| 프라이버시 | 해당 없음 | 없음 (감시) | **근무자가 편집 후 제출** |
| 직원 반발 | 없음 | 높음 | 낮음 |
| 지원금 증빙 | 별도 작업 필요 | 해외 서비스라 어려움 | **자동 리포트** |

### 핵심 셀링포인트

```
1. 지원금 증빙 - "Offio 쓰면 정부 지원금 받을 수 있습니다"
2. 모니터링 + 프라이버시 - "증빙은 하되, 감시는 아닙니다"
3. 비용 < 지원금 - "도입하면 오히려 돈이 남습니다"
```

## BM 요약

### 수익 구조

```
초기 구축비 (1회) + 월 이용료 (반복)
```

### 플랜

| 플랜 | 대상 | 인당 월 |
|------|------|---------|
| Lite | 기존 HR 있음, 증빙만 필요 (10명 제한) | 1.5만원 |
| Standard | 지원금 연계 중요 | 2.5만원 |
| Enterprise | HR 통합 원함 (휴가, 조직관리 포함) | 3.5만원 |

| 구분 | 금액 |
|------|------|
| 구축비 | 350만원 |

### 추가 수익원

- 지원금 신청 대행 (성공보수 10%)
- 커스텀 리포트 개발

## 초기 목표 (6개월)

### 비용 구조

**초기 개발비 (MVP 완성까지)**

| 항목 | 금액 |
|------|------|
| 개발 인건비 (2개월) | 2,000만원 |
| 서버/인프라 | 100만원 |
| 기타 | 100만원 |
| **합계** | **약 2,200만원** |

**월 운영비 (운영 단계)**

| 항목 | 금액 | 비고 |
|------|------|------|
| 서버/인프라 | 50만원 | AWS/클라우드 |
| 인건비 (운영/CS) | 300만원 | 1인 |
| 인건비 (영업) | 250만원 | 1인 |
| 마케팅/광고 | 50만원 | 콘텐츠, 소규모 광고 |
| 기타 | 50만원 | 툴, 예비비 |
| **합계** | **700만원** |  |

### 손익분기점

**월 700만원 커버 기준**
- 월 반복매출로만: 700만원 ÷ 2.5만원 = **280명**
- 평균 20명/사 기준 → **14개사**

### 목표 고객

| 지표 | 목표 |
|------|------|
| 고객사 | 15개사 |
| 총 사용자 | 300명 (평균 20명/사) |
| 구축비 매출 | 5,250만원 (350만원 × 15) |
| 월 반복 매출 | 750만원 (300명 × 2.5만원) |

### 마일스톤

| 시점 | 목표 |
|------|------|
| 2개월 | MVP 완성, 베타 고객 1개사 |
| 4개월 | 정식 출시, 고객 3개사 |
| 8개월 | 고객 8개사, 월 매출 400만원 |
| 12개월 | 고객 15개사, 월 매출 750만원 (손익분기) |
